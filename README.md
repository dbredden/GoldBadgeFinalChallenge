# Gold Badge Final Challenge

**This C# Solution is comprised of 3 individual console applications that require unit testing, usage of CRUD methods, a ProgramUI, and much more. 

## Project 01 - A Cafe Menu Application
This app allows a fictious cafe manager to add, delete, and see all items in a menu list. 

## Project 02 - An Insurance Claims Application
This app allows a fictious insurance claim agent to see all claims that are submitted, the ability to take the next claim in a queue, and to enter a brand new claim. 

## Project 05 - A Greeting Email Application
This app is made for a fictious administrator to see all customers in a customer database, and see all customers in alphabetical order, along with the ability to create, update, and delete existing customers.
